This is the CaringCookie Project. A project which was done for university.

CaringCookie is a plattform for students to exchange stuff in an eco friendly way.

To get started:

1. Download Python
2. Pull the GitHub project
3. To download all the dependencies: "pip install virtualenv"
5. To activate the virtual env open a terminal in the projects directory and enter: "source env/bin/activate"
6. Install necessary dependencies: "pipenv sync"
7. Start developing or look at the current state with: "python manage.py runserver" :)
